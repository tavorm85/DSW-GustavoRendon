Estructura básica de HTML5


HTML5 es la versión 5 del lenguaje HTML. Aquí mostraremos un documento escrito en HTML y su estructura básica:



En donde:

  • !DOCTYPE Html: Es la declaración del tipo de documento que se está mostrando. Este variará según el tipo de documento.

  • html: Una vez se declare el tipo de documento, debemos indicar que el documento se inicia, por tal motivo abrimos para iniciar y cerramos finalmente cuando terminemos el documento; por su parte el atributo lang, significa el idioma en el que está escrito.

  • head: Esta será la cabecera en donde se colocarán los metadatos de la página, además del código JavaScript y el CSS que el navegador va a utilizar.

  • title: Este metadato especial lo que hace es proporcionarnos el título de la página. Se recomienda que sea parecido al H1 del documento y a la URL de la página.

  • link: Usada para enlazar JavaScript y CSS con el documento actual HTML.

  • meta: Define los metadatos que no pueden ser definidos usando otros elementos de HTML. .

  • style: Es una etiqueta de estilo usada para escribir CSS interno.
